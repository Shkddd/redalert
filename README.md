# 红色警戒 网页版 + 超级冒险家

两个纯前端 HTML5 游戏，单文件无外部依赖，打开浏览器即玩。

## 文件

| 文件 | 说明 | 操作 |
|------|------|------|
| `index.html` | 红色警戒风格 RTS 游戏 | 双击打开 |
| `mario.html` | 超级马里奥风格平台跳跃游戏 | 双击打开 |

## 红色警戒 🎮

- RTS 即时战略，建造基地、采集资源、训练部队、抵御敌军
- 建筑：发电厂、矿石精炼厂、兵营、战车工厂、炮塔
- 单位：步兵、坦克、吉普车、矿车、修理车
- 操作：左键选择，右键移动/攻击，侧边栏建造

## 超级冒险家 🏃

- 横版平台跳跃闯关
- 特色操作：二段跳、蹭墙跳、下水管进密室
- ← → 移动，空格/↑ 跳跃，↓ 进入水管
- 收集金币，踩敌人，到达旗杆通关
